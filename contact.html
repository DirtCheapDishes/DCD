E-mail

For and questions, comments, suggestions, or concerns, please feel free to e-mail our team at DirtCheapDishes@gmail.com.

In order for us to provide you with the most prompt and helpful reply as possible, please make your message detailed and concise. Note that e-mails we send, like any other organization, may end up in your Spam folder. Be sure to check!

GitHub

"Global recipes" are recipes that are in a database for everyone to see, across all of DCD's available devices. To suggest a new one use the navigation menu on the top and bottom of every webpage on our site and click "Suggest a Recipe " or click here.

Users with GitHub accounts can contribute to DCD in a special way! To ask us to do something like fix a bug, add a recipe, or create a new feature, navigate to the the issues tab of DCD's GitHub repository and click "New Issue". Then, select the template that fits your needs!

GitHub users can help influence the apps' development directly by coding themselves! You may do this using Pull Requests. For more information on Pull Requests, read GitHub's help article.
